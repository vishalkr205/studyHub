# 📚 StudyHub

> A collection of chapter-wise PYQs, notes, important questions, and study resources to make exam preparation simple, organized, and accessible.

---

## 🎯 About

**StudyHub** is an open-source repository designed to help students prepare effectively for exams. It provides well-organized study materials, including chapter-wise notes, previous year questions (PYQs), important questions, and additional learning resources in one place.

---

## ✨ Features

* 📖 Chapter-wise Notes
* 📝 Previous Year Questions (PYQs)
* ⭐ Important Questions
* 📚 Study Materials and Resources
* 🔍 Easy Navigation by Subject and Chapter
* 🚀 Regularly Updated Content
* 💡 Beginner-Friendly Structure

---

## 🚀 Usage

1. Open the repository.
2. Select your subject.
3. Navigate to the required chapter.
4. Access Notes, PYQs, and Important Questions.
5. Use the materials for revision and exam preparation.
